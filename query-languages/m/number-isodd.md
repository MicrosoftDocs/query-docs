# Number.IsOdd

## Syntax

<pre>
Number.IsOdd(<b>number</b> as number) as logical  
</pre>

## About

Indicates if the value is odd. Returns `true` if `number` is an odd number, `false` otherwise.

## Example 1

Check if 625 is an odd number.

**Usage**

```powerquery-m
Number.IsOdd(625)
```

**Output**

`true`

## Example 2

Check if 82 is an odd number.

**Usage**

```powerquery-m
Number.IsOdd(82)
```

**Output**

`false`

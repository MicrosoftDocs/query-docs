# Number.BitwiseOr

## Syntax

<pre>
Number.BitwiseOr(<b>number1</b> as nullable number, <b>number2</b> as nullable number) as nullable number
</pre>
  
## About  
Returns the result of performing a bitwise "Or" between `number1` and `number2`.

# Value.Compare

## Syntax

<pre>
Value.Compare(<b>value1</b> as any, <b>value2</b> as any, optional <b>precision</b> as nullable number) as number
</pre>

## About

Returns -1, 0, or 1 based on whether the first value is less than, equal to, or greater than the second.

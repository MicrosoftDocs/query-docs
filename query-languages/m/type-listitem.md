# Type.ListItem

## Syntax

<pre>
Type.ListItem(<b>type</b> as type) as type
</pre>
  
## About

Returns an item type from a list `type`.

## Example 1

Find item type from the list `{number}`.

**Usage**

```powerquery-m
Type.ListItem(type {number})
```

**Output**

`type number`

# Text.Length

## Syntax

<pre>
Text.Length(<b>text</b> as nullable text) as nullable number
</pre>
  
## About

Returns the number of characters in the text `text`.

## Example 1

Find how many characters are in the text "Hello World".

**Usage**

```powerquery-m
Text.Length("Hello World")
```

**Output**

`11`

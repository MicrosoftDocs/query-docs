# Order.Type

## Definition

Specifies the direction of sorting.

## Allowed values

|Name|Value|Description|
| ------- | --- | ----------- |
|**Order.Ascending**|0|Sorts the values in ascending order.|
|**Order.Descending**|1|Sorts the values in descending order.|

* [Table functions](table-functions.md)

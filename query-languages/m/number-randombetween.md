# Number.RandomBetween

## Syntax

<pre>
Number.RandomBetween(<b>bottom</b> as number, <b>top</b> as number) as number
</pre>
  
## About

Returns a random number between `bottom` and `top`.

## Example 1

Get a random number between 1 and 5.

**Usage**

```powerquery-m
Number.RandomBetween(1, 5)
```

**Output**

`2.546797`

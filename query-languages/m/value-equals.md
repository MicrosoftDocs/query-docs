# Value.Equals

## Syntax

<pre>
Value.Equals(<b>value1</b> as any, <b>value2</b> as any, optional <b>precision</b> as nullable number) as logical
</pre>

## About

Returns true if value `value1` is equal to value `value2`, false otherwise.

# Value.Divide

## Syntax

<pre>
Value.Divide(
    <b>value1</b> as any,
    <b>value2</b> as any,
    optional <b>precision</b> as nullable number
) as any
</pre>
  
## About

Returns the result of dividing `value1` by `value2`. An optional `precision` parameter may be specified, by default `Precision.Double` is used.

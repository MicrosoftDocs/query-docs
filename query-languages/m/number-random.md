# Number.Random

## Syntax

<pre>
Number.Random() as number
</pre>
  
## About

Returns a random number between 0 and 1.

## Example 1

Get a random number.

**Usage**

```powerquery-m
Number.Random()
```

**Output**

`0.919303`

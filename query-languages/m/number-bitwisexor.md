# Number.BitwiseXor

## Syntax

<pre>
Number.BitwiseXor(<b>number1</b> as nullable number, <b>number2</b> as nullable number) as nullable number
</pre>

## About

Returns the result of performing a bitwise "XOR" (Exclusive-OR) between `number1` and `number2`.

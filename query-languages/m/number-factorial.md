# Number.Factorial

## Syntax

<pre>
Number.Factorial(<b>number</b> as nullable number) as nullable number
</pre>

## About

Returns the factorial of the number `number`.

## Example 1

Find the factorial of 10.

**Usage**

```powerquery-m
Number.Factorial(10)
```

**Output**

`3628800`

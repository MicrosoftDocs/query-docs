# List.Count

## Syntax

<pre>
List.Count(<b>list</b> as list) as number
</pre>

## About

Returns the number of items in the list `list`.

## Example 1

Find the number of values in the list {1, 2, 3}.

**Usage**

```powerquery-m
List.Count({1, 2, 3})
```

**Output**

`3`

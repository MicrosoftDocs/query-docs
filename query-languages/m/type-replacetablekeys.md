# Type.ReplaceTableKeys

## Syntax

<pre>  
Type.ReplaceTableKeys(<b>tableType</b> as type, <b>keys</b> as list) as type
</pre>
  
## About  
Returns a new table type with all keys replaced by the specified list of keys.

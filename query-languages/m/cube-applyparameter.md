# Cube.ApplyParameter

## Syntax

<pre>
Cube.ApplyParameter(cube as table, parameter as any, optional arguments as nullable list) as table  
</pre>
  
## About  
  
Returns a cube after applying `parameter` with `arguments` to `cube`.

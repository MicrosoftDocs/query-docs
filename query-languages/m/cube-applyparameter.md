# Cube.ApplyParameter

## Syntax

<pre>
Cube.ApplyParameter(
    <b>cube</b> as table,
    <b>parameter</b> as any,
    optional <b>arguments</b> as nullable list
) as table
</pre>

## About

Returns a cube after applying `parameter` with `arguments` to `cube`.
